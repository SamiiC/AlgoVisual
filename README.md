# Pathfinding Visualiser
Visualiser for multiple different algorithms including:
- A* Pathfinding
- Dijkstra's Pathfinding
- Depth First Search
- Breadth First Search
- Randomized Prim's Algorithm
- Recursive Backtracking for use in maze generation

See the [wiki](https://github.com/SamiiC/AlgoVisual/wiki) for a demo and details for the app

## How to build + start the app

Clone the project and run:

`npm install` to install dependancies

`npm build` to build the app

`npm preview` to then start app on localhost
